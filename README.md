# event_detector

> [!NOTE]  
> The event detector is a generic, modular, and data-driven event detection framework, developed in the context of Cooperative Intelligent Transport Systems (C-ITS). The framework is being open-sourced as part of our publication *Event Detection in C-ITS: Classification, Use Cases,
and Reference Implementation* at the *2025 Uni-DAS e.V. Workshop Fahrerassistenz und automatisiertes Fahren*.  
> 🔜 Source code and documentation will be made available here by April 2025.
